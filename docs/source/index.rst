paccs : Python Analysis of Colloidal Crystal Structures
=======================================================

The user guide contains information useful for applying paccs to solve various
problems in colloidal crystal science.  Supplementary to this is the API documentation, 
which describes in detail every option that can be provided to various paccs routines.  A
few other miscellaneous pages describe other features of the package that don't
fit anywhere else.
